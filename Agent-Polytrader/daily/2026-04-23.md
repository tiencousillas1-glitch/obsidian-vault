### Dashboard v8 — Bug fixes masivos
Tien reportó inconsistencias numéricas en el dashboard. Análisis exhaustivo reveló 6 bugs:

1. **6 posiciones duplicadas** — FC Barcelona, Ratiopharm Ulm, Milwaukee, Copenhagen Over, Cleveland, Pistons aparecían en BOTH Polymarket API y resolved.json, inflando cifras
2. **"Ganadas" $7.87 tile vs $9.91 barra** — tile usaba wonPnl, barra usaba wonCurrent + resolvedOnlyPnl
3. **"Invertido" $15.70 vs "Activas" $5.79** — invertido sumaba won+active, confuso
4. **Trades 14 vs pestaña "Todas 16"** — stats.bought (303) no es trades reales
5. **Deploy 18% vs 33%** — dos fórmulas distintas para el mismo número
6. **P&L -$18.20 vs -$18.24** — rounding inconsistente

**Solución:** Reescribí server.js para que calcule TODAS las stats en backend (zero calculations en frontend). Dedup automático de posiciones. Una sola fuente de verdad.
